# wip-cortex-local

A collection of skills, workflows, tools, integrations, and best practices for development with AI.

Updated frequently. Local-first. Credited where the work is not original.

## structure

- `design/` — UI, motion, accessibility, and visual implementation patterns
- `design-systems/` — reference design systems used only when explicitly needed
- `dev-tools/` — overlays, annotation tools, tuning tools, and local dev helpers
- `workflows/` — agent workflows, review patterns, conventions, and setup guides
- `marketing/` — marketing skills and related tools

# Default Stack

## Frontend
- Next.js (app router)
- Tailwind CSS
- Framer Motion
- shadcn/ui components (adapted, not stock)

## Backend
- Supabase (auth, postgres, edge functions, realtime)
- Vercel (hosting, CI/CD)

## Tooling
- Bun (package manager + runtime)
- TypeScript (strict)
- ESLint + Prettier

## Design
- Figma (design)
- OKLCH color system
- Dark mode first

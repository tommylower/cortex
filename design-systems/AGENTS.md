- **swiss-design** — Swiss International Style design system. grid-first, grotesque typography, restrained color, opacity hierarchy, one accent. only when user asks for Swiss/modernist editorial style
- **infrastructure-design** — institutional technical marketing-site system. dark ink/paper contrast, precise hairlines, square geometry, sparse green signal, diagrammatic motion. only when explicitly requested
- **report-design** — report marketing-site system. cream paper surface, ink frame, red signal/action accent, square cards, operational section rhythm. only when explicitly requested

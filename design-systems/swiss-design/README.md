# swiss-design

`swiss-design` is an opt-in reference system for Swiss International Style interfaces: rigorous grids, clear grotesque typography, restrained color, and whitespace as structure.

Use it when the project explicitly asks for Swiss, modernist, editorial, grid-first, or Helvetica-like design language.

## what it contains

- `SKILL.md` — operating brief, trigger conditions, core rules, type guidance, color defaults, layout pattern, and preflight checklist

This system is intentionally compact. It is useful as a starting point or style lens rather than a full runtime extraction.

## how to use it

Start with the grid and hierarchy before choosing decorative details. Pick one accent, keep body text narrow, use opacity for hierarchy, and make mobile collapse cleanly to one column.

With waveframe, this can seed a new project-specific design system. Once real components, tokens, and implementation details exist, promote those decisions into deeper reference files such as `tokens.md`, `composition.md`, `components.md`, and `platform-mapping.md`.

# Nothing Design System — Voice

## Tone

Precise, terse, and mechanical. Interface copy should feel like an instrument label or technical manual, not a marketing campaign.

## Sentence structure

- Short sentences.
- Direct verbs.
- No exclamation marks.
- Use bracketed mono status labels sparingly: `[SAVED]`, `[ERROR]`, `[LOADING]`.

## Vocabulary

| Use | Avoid |
| --- | --- |
| `connected` | `all set` |
| `saved` | `awesome` |
| `sync failed` | `something went wrong` |
| `over limit` | `oops` |

## Microcopy

Buttons use direct commands: `Save`, `Connect`, `Delete`, `Retry`. Labels may be uppercase in UI when using Space Mono.

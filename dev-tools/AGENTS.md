# dev-tools

overlays, annotation tools, tuning tools, and local dev helpers that get installed into a project.

## index

- **deadcode** — find and remove unused files, exports, dependencies, and types. lists findings with reasoning before deleting
- **agentation** — install the Agentation visual feedback toolbar in a Next.js project. by Dennis Jin and Benji Taylor
- **agentation-self-driving** — autonomous design critique mode using the Agentation toolbar. requires agentation + agent-browser
- **dialkit** — install the DialKit floating control panel for tuning animations and visual properties. by Josh Puckett
- **interface-kit** — install the Interface Kit visual design overlay in a Next.js project. by Josh Puckett
- **pretext** — pure JS/TS library for fast, accurate multi-line text measurement and layout without DOM reflow. by Cheng Lou. use whenever working on typography rules, label overflow, virtualization, masonry, balanced text, shrink-wrap, async text layout-shift prevention

see ../AGENTS.md for the cortex layout and skill format.
